## Phase 3 — Fixes Applied

### Fix 1: Prowlarr URL injection (P0 — V-01)
- **File:** `sokoul-backend/src/services/prowlarr.rs`
- **Change:** Replaced `format!()` URL construction with `.query()` builder
- **Impact:** Prevents URL corruption from special characters in movie titles

### Fix 2: HTTP client timeouts (P0 — H-02, P1 — V-04)
- **File:** `sokoul-backend/src/main.rs`
- **Change:** Reduced global timeout from 30s → 10s (per CLAUDE.md policy), added `connect_timeout(5s)`
- **Impact:** Fail-fast on unreachable hosts, compliance with documented policy

### Fix 3: CastMember type alignment (P1 — A-05)
- **File:** `sokoul-desktop/src/shared/types/index.ts`
- **Change:** Made `character` optional, added `order: number`, removed dead `profilePath` fallback
- **Impact:** Matches Rust CastMember model, prevents null crashes

### Fix 4: ListItem phantom field removal (P1 — A-02, A-03)
- **File:** `sokoul-desktop/src/shared/types/index.ts`
- **Change:** Removed `backdropUrl`, `year`, `rating` (never sent by API); added `sortOrder: number`
- **Impact:** Eliminates false type safety, enables future drag-to-reorder

### Fix 5: UserList traktListId (P1 — A-04)
- **File:** `sokoul-desktop/src/shared/types/index.ts`
- **Change:** Added `traktListId?: string`
- **Impact:** Frontend can now distinguish Trakt-synced lists

### Fix 6: React Query staleTime (P1 — P-01)
- **File:** `sokoul-desktop/src/app/main.tsx`
- **Change:** Added global `staleTime: 5min`, `refetchOnWindowFocus: false`
- **Impact:** Eliminates dozens of unnecessary API refetches on desktop

### Fix 7: Cascading reference updates
- **Files:** `MyListsPage.tsx`, `useHomePersonalized.ts`, `CastSection.tsx`
- **Change:** Removed references to deleted phantom fields
- **Impact:** Clean compilation, no runtime undefined access
